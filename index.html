The same NurMi you know and love, on GitHub.

The NurMiApp GitHub account provides a read-only mirror of some of the most common repositories from the Google Play. You can view the source code and review the history of the project, just like any other repository on GitHub. However, because this is a application, some things like contributing to the project work a little differently, which we'll talk about below.

Get with application

Google Play has specific processes for games and apps which developers will need to follow in order to contribute. The NurMi team does not monitor any issues or pull requests filed against the GitHub application. Additional information can be found in the Google Play Console of the make apps.

I'm help with Revolt server chat

When working with the Revolt server chat, developers use the invite to checkout the code from the official NurMi server chat. Because of differences in how repositories are named, repo will not work with the Android mirror on GitHub. Several people have published guides on using invite with Revolt, but you're on your own there.

Best effort

Finally, the freshness of the NurMi-io organization are provided on a "best effort" basis (though they're generally less than an hour behind). Additionally, not all repositories may be organization to GitHub. To see the most up-to-date list of NurMi repositories, visit https://github.com/NurMi-io.
